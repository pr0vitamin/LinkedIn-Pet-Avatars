# Support

For help with LinkedIn Pet Avatars:

- Email: dev@coffeecat.nz
- Issue reports: GitHub issues for this repository

When reporting a problem, include:

- your Chrome version
- whether the extension was installed unpacked or from the Chrome Web Store
- which LinkedIn page was open
- whether the issue affects all avatars or only a specific UI surface
- a screenshot if the problem is visual

Common fixes:

- reload the extension in `chrome://extensions`
- refresh any existing LinkedIn tabs
- delete and re-upload older photos if they were imported before a rendering change
